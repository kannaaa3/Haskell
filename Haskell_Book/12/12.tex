    \section{Lifted and unlifted types}
    lifted: * (all standard lifted types), unlifted : \#

    \begin{Def}[Lifted types]
        A lifted type, which includes any datatype you could define yourself, is any
        that can be inhabited by \textit{bottom}. Represented by a pointer.
    \end{Def}
    \begin{Def}[Unlifted type]
        Unlifted types are any type which \textit{cannot} be inhabited by bottom (native machine
        types and raw pointers).
    \end{Def}

    Newtypes are special case.
